# FreeScout Inbox Folder

Adds a unified Inbox folder at the top of the FreeScout sidebar showing all active conversations across all mailboxes in one place.

## Features

- Unified inbox at the top of the sidebar
- Shows all active conversations regardless of mailbox
- Real-time count badge

## Requirements

- FreeScout v1.8+
- PHP 8.0+

## Installation

1. Upload the `InboxFolder` folder to `Modules/`
2. Go to Manage → Modules and activate it
